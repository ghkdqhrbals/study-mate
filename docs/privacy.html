총칙

StudyMate는 사용자가 직접 입력한 OpenAI API 키로 학습 질문 생성과 답변 채점을 이용하는 무료 학습 앱입니다. StudyMate는 별도의 자체 백엔드 서버를 운영하지 않으며, 개인정보를 판매하지 않습니다.

이 개인정보 처리방침은 개인정보보호위원회의 「개인정보 처리방침 작성지침」(2025.4.)에서 안내하는 명확성, 투명성, 접근성 원칙을 참고하여 작성되었습니다.

1. 개인정보의 처리 목적

StudyMate는 다음 목적을 위해 필요한 범위에서 정보를 처리합니다.

  • 학습 주제, 난이도, 질문 간격 등 앱 설정 저장
  • OpenAI API를 통한 학습 질문 생성 및 답변 채점
  • 답변 초안, 학습 기록, 점수, 피드백, 해설, 주제별 통계 제공
  • iCloud 동기화 사용 시 동일 Apple ID 기기 간 학습 데이터 동기화
  • 알림 허용 시 학습 질문 알림 제공
  • 디버깅 모드 사용 시 오류 확인을 위한 앱 로그 표시

2. 처리하는 개인정보 항목

StudyMate는 계정 가입을 요구하지 않습니다. 앱 사용 과정에서 다음 정보가 사용자의 기기 또는 사용자가 선택한 외부 서비스에 저장될 수 있습니다.

구분 항목 처리 위치
앱 설정 학습 주제, 난이도, 프롬프트, 질문 간격, 언어, 알림음, 기록 보관 개수 기기 내 저장, iCloud 동기화 사용 시 CloudKit
OpenAI 설정 사용자가 입력한 일반 OpenAI API 키, 모델 설정 기기 내 저장, iCloud 동기화 사용 시 CloudKit
학습 데이터 질문, 힌트, 답변 초안, 제출 답변, 점수, 피드백, 해설, 학습 기록, 주제별 통계 기기 내 저장, iCloud 동기화 사용 시 CloudKit, AI 기능 사용 시 OpenAI API
진단 정보 디버깅 모드에서 표시되는 앱 동작 로그 기기 내 저장

OpenAI Admin API 키는 기기 로컬에만 저장되며 iCloud로 동기화하지 않습니다.

3. 개인정보의 처리 및 보유 기간

앱 설정, API 키, 학습 기록 및 통계는 사용자가 앱을 사용하는 동안 기기 내에 보관됩니다. 학습 기록은 앱 설정의 최대 기록 개수에 따라 보관되며, 사용자는 앱 안에서 개별 기록 또는 전체 기록을 삭제할 수 있습니다.

앱을 삭제하면 iOS 동작에 따라 기기 내 로컬 데이터가 삭제됩니다. iCloud 동기화를 사용한 경우 iCloud에 동기화된 데이터는 Apple ID 및 iCloud 설정의 영향을 받을 수 있습니다.

4. 개인정보의 파기 절차 및 방법

사용자가 앱에서 기록을 삭제하면 해당 기록은 앱의 로컬 저장소에서 삭제됩니다. 앱 삭제 시 로컬 앱 데이터는 iOS의 앱 데이터 삭제 방식에 따라 제거됩니다. iCloud 동기화 데이터는 Apple이 제공하는 iCloud 관리 기능을 통해 관리될 수 있습니다.

5. 제3자 제공, 처리위탁 및 국외 이전

StudyMate는 자체적으로 개인정보를 판매하거나 광고 사업자에게 제공하지 않습니다. 다만 사용자가 선택한 기능을 제공하기 위해 다음 외부 서비스를 이용할 수 있습니다.

서비스 이용 목적 전송 또는 저장될 수 있는 정보
OpenAI API 학습 질문 생성, 답변 채점, 피드백 및 해설 생성 학습 주제, 질문, 힌트, 사용자가 입력한 답변, 관련 프롬프트
Apple iCloud / CloudKit 사용자가 켠 경우 동일 Apple ID 기기 간 설정, API 키, 학습 상태, 기록 동기화 앱 설정, 일반 OpenAI API 키, 질문, 답변 초안, 기록, 점수, 피드백, 해설
Apple Push Notification service 알림 허용 시 학습 질문 알림 전달 알림 전달에 필요한 기기 및 알림 정보

OpenAI API와 Apple 서비스는 국외에서 운영될 수 있으며, 각 서비스의 약관 및 개인정보 처리방침이 적용됩니다. 사용자는 OpenAI API 키를 입력하지 않거나 iCloud 동기화를 끄는 방식으로 해당 기능 사용을 제한할 수 있습니다.

6. 개인정보 자동 수집 장치

StudyMate 앱은 자체 광고 추적 쿠키나 웹 추적기를 사용하지 않습니다. GitHub Pages로 제공되는 소개 페이지는 GitHub가 제공하는 정적 호스팅 환경에서 제공되며, GitHub의 서비스 운영에 필요한 로그가 처리될 수 있습니다.

7. 이용자의 권리와 행사 방법

사용자는 앱에서 자신의 학습 기록을 확인하고 삭제할 수 있으며, iCloud 동기화를 끄거나 알림 권한을 iOS 설정에서 변경할 수 있습니다. OpenAI API 키는 앱 설정에서 변경하거나 삭제할 수 있습니다.

OpenAI 또는 Apple에 저장·처리되는 정보에 관한 권리는 각 서비스의 계정 설정 및 개인정보 요청 절차를 통해 행사할 수 있습니다.

8. 개인정보의 안전성 확보조치

StudyMate는 개인정보 보호를 위해 다음 조치를 적용합니다.

  • 별도 자체 서버를 운영하지 않고 기기 중심으로 데이터를 저장
  • OpenAI Admin API 키는 iCloud로 동기화하지 않음
  • iCloud 동기화는 사용자가 켠 경우에만 사용
  • 디버깅 로그는 앱 내부에서만 확인 가능하며 최대 1,000개로 제한
  • 앱과 외부 서비스 간 통신은 운영체제의 표준 보안 통신을 사용

9. 14세 미만 아동의 개인정보 처리

StudyMate는 14세 미만 아동을 대상으로 하지 않습니다. 14세 미만 아동의 개인정보가 앱을 통해 처리되었다고 판단되는 경우 아래 문의처로 연락해 주세요.

10. 자동화된 결정

StudyMate의 채점 결과는 OpenAI API를 통해 생성되는 학습 피드백입니다. 이는 사용자의 학습 참고용으로 제공되며, 법적 효과나 이와 유사하게 중대한 영향을 미치는 자동화된 결정에는 해당하지 않습니다.

13. 개인정보 처리방침의 변경

이 개인정보 처리방침은 법령, 서비스 또는 앱 기능 변경에 따라 업데이트될 수 있습니다. 변경된 내용은 이 페이지에 게시합니다.
