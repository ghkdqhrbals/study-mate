StudyMate

In the AI era, better output still starts with what you know. StudyMate is a free macOS menu bar AI tutor that helps you build that knowledge a little at a time.

Source Code Releases

The software is completely free. To use it, set your own OpenAI API key.

  • Lives in the menu bar as a quiet AI teacher.
  • Choose a topic and difficulty, then get questions on a schedule.
  • Answer and get graded so you can quickly check what you know.
  • Free and open source, with API usage billed through your own OpenAI account.
