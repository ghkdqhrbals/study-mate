StudyMate

In the AI era, better output still starts with what you know. StudyMate is a free macOS menu bar AI tutor that helps you build that knowledge a little every day.

v1.0.6 MIT License
Download DMG GitHub

The app is free. To use it, set your own OpenAI API key.

What it does

  • Get questions by topic and difficulty.
  • Answer and receive a score, feedback, and explanation.
  • Review records and statistics to see what you know.

Why I made it

I use AI every day, and I realized that the more I understand a subject, the sharper my prompts become and the better the output gets. I built StudyMate to make that habit easier to keep and share.

Latest version: v1.0.6. Released under the MIT License. API usage is billed through your own OpenAI account.
