StudyMate

AI 시대에도 좋은 결과물은 내가 가진 지식에서 시작됩니다. StudyMate는 그 지식을 매일 조금씩 쌓도록 돕는 무료 macOS 메뉴바 AI 튜터입니다.

v1.0.7 MIT License
DMG 다운로드 GitHub

앱은 무료입니다. 사용하려면 본인의 OpenAI API key가 필요합니다.

무엇을 하나요

  • 원하는 주제와 난이도로 질문을 받습니다.
  • 답변하면 AI가 점수, 피드백, 해설을 제공합니다.
  • 기록과 통계로 내가 얼마나 알고 있는지 확인합니다.

만든 이유

AI를 계속 쓰다 보니, 내가 잘 아는 만큼 질문이 선명해지고 결과물의 해상도도 높아진다는 걸 느꼈습니다. 그 감각을 더 쉽게 유지하고 나누고 싶어서 만들었습니다.
