StudyMate

AI 시대에도 좋은 산출물은 스스로 가진 지식에서 시작됩니다. StudyMate는 그 지식을 조금씩 쌓게 도와주는 무료 macOS 메뉴바 AI 튜터입니다.

Source Code Releases

소프트웨어는 완전 무료입니다. 사용하려면 본인의 OpenAI API key만 설정하면 됩니다.

  • 메뉴바에서 조용히 실행되는 AI 선생님.
  • 주제와 난이도를 정하면 주기적으로 질문합니다.
  • 답변 후 채점으로 내가 아는지 바로 확인합니다.
  • 무료 오픈 소스로 사용할 수 있고, API 비용은 사용자의 OpenAI 계정에서 처리됩니다.

안녕하세요.

평소에 AI를 계속 쓰다 보니, 결국 잘 아는 만큼 질문이 선명해지고 결과물의 해상도도 높아진다는 걸 느꼈습니다. 그 경험을 다 같이 나누고 싶어서 StudyMate를 만들었습니다.
