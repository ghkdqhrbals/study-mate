StudyMate

AI 시대에도 좋은 산출물은 스스로 가진 지식에서 시작됩니다. StudyMate는 그 지식을 조금씩 쌓게 도와주는 macOS 메뉴바 AI 튜터입니다.

Source Code Releases

완전 무료. OpenAI API 키만 직접 설정해서 사용합니다.

  • 메뉴바에서 조용히 실행되는 AI 선생님.
  • 주제와 난이도를 정하면 주기적으로 질문합니다.
  • 답변 후 채점으로 내가 아는지 바로 확인합니다.
  • 무료로 사용할 수 있는 오픈 소스 앱입니다.
